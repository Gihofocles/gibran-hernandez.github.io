Vending Machine

Using Labview and Arduino

← Back to projects

General description

This project involves the design and implementation of an automated vending machine system using LabVIEW as the main control interface and Arduino as the hardware controller. The system integrates both software and hardware components to simulate the complete operation of a real vending machine.

The LabVIEW graphical interface allows the user to select products, display their prices, and monitor system status in real time. It communicates with the Arduino microcontroller via serial communication to control the dispensing mechanisms, stepper motors, and to receive input signals from sensors and buttons.

Detailed description

The program is structured using a state machine architecture, managing processes such as product selection, payment verification, item dispensing, and inventory updates. The interface also provides visual feedback, showing the selected item, remaining balance, and transaction status.

Additional features include errors, such as insufficient payment or product unavailability, and expandable functionality for integration with other payment systems or components like LCD displays, LEDs, and buzzers.

The system includes a temperature sensor and a cooling fan to ensure that items are stored at safe temperatures. The application was developed as a standalone executable (.exe), allowing it to run independently of the LabVIEW development environment.
