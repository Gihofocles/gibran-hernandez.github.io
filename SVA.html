Sistema de visión artificial

Clasificación de productos para control de calidad

← Volver a Proyectos

Descripción General

A computer vision system with actuators was developed for the automatic classification of products. Its implementation involved segmentation algorithms based on color and shape, using the RGB and HSV models, as well as the image signature and Hu invariant moments for pattern detection.

To ensure proper image capture, a controlled environment was established with artificial lighting using lamps. Additionally, actuators were integrated to physically separate compliant objects from defective ones.

The system incorporated databases for storing templates and results and enabled the automatic generation of Excel reports documenting the characteristics of each analyzed object.

This project strengthened skills in computer vision, control, data analysis, and multidisciplinary teamwork, resulting in a robust system applicable to industrial automation processes.

Tecnologías Utilizadas

MATLAB Arduino Artificial vision
