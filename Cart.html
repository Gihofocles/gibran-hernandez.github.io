Design and Development of an Active Suspension

Using Esp32 and PID

← Back to projects

General description

As the final project for the Classical Control course, this work integrates most of the theoretical and practical concepts studied throughout the semester. The project focuses on the design and implementation of an active suspension system for a miniature vehicle, aiming to analyze the stability, response, and behavior of a controlled dynamic system.

The system maintains a fixed distance between the car’s chassis and the ground using servomotors connected to the wheel axles and an ultrasonic sensor that measures the height in real time. A PID controller implemented on an ESP32 microcontroller regulates the servomotors to keep the vehicle at the desired height, even when traveling over uneven or irregular surfaces.

The main objective is to develop a fully functional and stable suspension system, minimizing oscillations during transient states and maintaining the distance within a user-defined range.

Detailed description

The theoretical development began with the derivation of the transfer function of the suspension system, modeled as a second-order dynamic system. This model was used to simulate the system behavior and design the PID controller parameters using the Ziegler–Nichols tuning method, which provided optimal proportional, integral, and derivative gains based on the tangent line of the system’s step response.

The simulation and analysis were performed using MATLAB and Simulink, allowing evaluation of the transient response and controller performance. The results showed that the system achieved a short response time and negligible overshoot, ensuring a stable and precise control of the vehicle’s height.

For the control implementation, the ESP32 was programmed in C++ (Arduino environment) to process the distance measurements and adjust the servomotor positions in real time. The algorithm computes the error between the measured and desired distances and applies the PID control law to maintain a stable suspension level.

The mechanical assembly included metal joints and screws to reinforce the chassis, while electronic components were connected to the ESP32. The control algorithm, programmed in C++ for Arduino, computes the error between the measured and desired distances and updates the servomotor angles accordingly to stabilize the car body.
