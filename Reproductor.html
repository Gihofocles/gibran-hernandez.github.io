General description

The project consists of the development of a music player application built in LabVIEW. It is designed to play WAV audio files, providing users with essential playback controls such as play, pause, stop, next, and previous. The interface also features a real-time frequency spectrum display, allowing users to visualize the audio signal while it is playing. Additionally, the system supports playlist management, enabling multiple tracks to be loaded and controlled within an intuitive graphical user interface.

Detailed description

This project implements a fully functional music player developed in LabVIEW, designed to play and manage WAV audio files. The system is structured using a state machine architecture, which efficiently handles the menu navigation, button interactions, and playback logic. The user interface displays detailed information for each track, including the file name, total duration, and a real-time progress bar indicating the playback position.

The music player features a playlist system organized by file directories, automatically detecting and listing all available audio files within each folder. Users can easily select and play any song, while the application supports autoplay functionality, seamlessly advancing to the next track once the current one finishes.

Additionally, the program includes a frequency spectrum display generated through a Fast Fourier Transform (FFT), providing a visual representation of the audio signal’s frequency content in real time. The combination of these features results in an interactive, dynamic, and user-friendly LabVIEW-based music player.
