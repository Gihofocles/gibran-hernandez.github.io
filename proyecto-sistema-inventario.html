Sistema de Control de Inventario

Solución backend para la gestión de stock en tiempo real.

← Volver a Proyectos

Descripción General

Este proyecto fue desarrollado para resolver el problema de la falta de seguimiento de inventario en un pequeño almacén. La aplicación permite a los usuarios registrar productos, actualizar cantidades, y generar reportes de stock bajo.

La API REST fue construida con Python y FastAPI, lo que garantiza una alta velocidad de respuesta. Se utilizó PostgreSQL para la base de datos debido a su fiabilidad y capacidad para manejar consultas complejas.

Desafíos y Soluciones

Uno de los principales desafíos fue asegurar la concurrencia, es decir, evitar que dos usuarios modificaran la misma cantidad de stock al mismo tiempo. Esto se solucionó implementando transacciones a nivel de base de datos para asegurar la atomicidad de las operaciones.

Tecnologías Utilizadas

Python FastAPI PostgreSQL Docker Pytest
